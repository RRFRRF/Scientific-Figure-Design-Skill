# Scientific Figure Prompt Generator

A meta-prompt template for generating high-quality AI drawing prompts for scientific figures.

## What is this?

This is a **prompt-for-prompts** — a structured template that helps you create effective prompts for AI image generators (like Midjourney, DALL-E, or Gemini) to produce publication-ready scientific diagrams.

## How to Use

1. Open `scientific_figure_prompt_template.md`
2. Fill in the `[USER INPUT]` section:
   - Select language
   - Choose reference type
   - Describe your visualization goal
   - Provide technical content
3. Delete options you don't need
4. Copy the entire file to your AI assistant (Gemini/Claude)
5. Get a tailored drawing prompt

## Features

- **3-Phase Process**: Understand → Think → Design
- **Structured Output**: Scene, elements, layout, colors, style specs
- **Publication-Ready**: Optimized for IEEE/Nature/ACM standards
- **Flexible Input**: Works with or without reference sketches

## Output

The template generates prompts with:
- Scene description
- Visual elements hierarchy
- Color palette with hex codes
- Technical specifications (4K, 300 DPI)
- Negative prompts for quality control
